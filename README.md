# 9x9quiz

このプロジェクトは、小学校2年生向けの九九かけ算クイズアプリです。
短いクイズをくりかえし遊ぶことで、九九の自然な定着を目指します。

## コアバリュー

本アプリのコアバリューは次です。

* 小学校2年生が「もういっかいやりたい」と感じて、自発的にくりかえし取り組むこと
* その反復の結果として、九九が自然に身につくこと

詳細は `docs/reference/core-value.md` を参照してください。
