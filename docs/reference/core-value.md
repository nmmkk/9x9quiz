# Core Value

Last updated: 2026-02-14

## Core Value Statement

* 小学校2年生が「もういっかい」と思って九九クイズを自発的に反復し、結果として九九が自然に定着する状態を作る。

## Product Principles

* Repeat First: 1回のセッションは短く、すぐ再挑戦できる導線を優先する。
* Simple Controls: 小2が一人で迷わず進める画面遷移と操作を優先する。
* Positive Feedback: 正解時は気持ちよさを強め、誤答時は責めずに次へ進ませる。
* Learning over Gimmicks: 演出は学習テンポを壊さない範囲で使う。
* Grade-2 Friendly Copy: ひらがな中心で短くやさしい文言を使う。

## Decision Filter

新しい改善案は、次を満たすほど優先する。

1. 反復意欲が上がるか。
2. 九九の定着に効くか。
3. 小2が一人で迷わず操作できるか。
4. 演出が学習テンポを壊さないか。

## Non-goal Guardrails

* 初期段階では複雑なルール追加より、反復体験の質向上を優先する。
* 競争性や難易度の急上昇で離脱を増やす要素は後回しにする。
* スコープ拡大より、既存体験の分かりやすさを優先する。
